SINAI RÜYA: DİJİTAL SERGİ / INDUSTRIAL DREAM: DIGITAL EXHIBITION
[TR] Proje Hakkında
Sınai Rüya, "Sinan At Biniyor" isimli edebi eseri referans alarak, ortaokul ve lise seviyesindeki öğrencilerin yazdığı kodlar aracılığıyla hayata geçirilmiş disiplinler arası bir dijital sergidir. Bu çalışma, endüstriyel mirasın soğuk yüzü ile çocuk ruhunun rüya dünyasını kod satırları arasında birleştirmeyi hedefler.

Detaylar
Kaynak Eser: Sinan At Biniyor (Sinan Rides a Horse)

Yöntem: Öğrenciler, kitaptaki betimlemeleri Python ve yaratıcı kodlama kütüphaneleri kullanarak görsel sanat eserlerine dönüştürmüştür.

Kapsam: Maden ocakları, buharlı makineler ve at figürleri, algoritmik sanat teknikleriyle (generative art) yeniden yorumlanmıştır.

Amaç: STEM eğitimini sanatla (STEAM) birleştirerek öğrencilerin teknik becerilerini sanatsal ifadeyle güçlendirmek.

[EN] About the Project
Industrial Dream is an interdisciplinary digital exhibition brought to life through codes written by middle and high school students, referencing the literary work titled "Sinan At Biniyor". This project aims to merge the industrial heritage's cold reality with the dream world of a child's spirit within lines of code.

Details
Source Work: Sinan At Biniyor (Sinan Rides a Horse)

Methodology: Students transformed descriptions from the book into visual artworks using Python and creative coding libraries.

Scope: Coal mines, steam engines, and horse figures are reinterpreted using algorithmic (generative) art techniques.

Objective: To strengthen students' technical skills with artistic expression by combining STEM education with Art (STEAM).

Teknik Araçlar / Technical Stack
Language: Python

Libraries: Processing (py.processing), Pillow, OpenCV

Methodology: Generative Art, Algorithmic Design
